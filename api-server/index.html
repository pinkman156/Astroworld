Astro Insights API Server

This server provides API endpoints for the Astro Insights application.

Available Endpoints:

GET /health

Health check endpoint to verify the server is running.

GET /api/geocode

OpenStreetMap geocoding endpoint.

POST /api/prokerala/token

Get Prokerala API OAuth token.

GET /api/prokerala/planet-position

Get planet positions from Prokerala API.

GET /api/prokerala/kundli

Get Kundli data from Prokerala API.

GET /api/prokerala/chart

Get chart data from Prokerala API.

POST /api/together/chat

Together AI API endpoint for chat completions.
